# typescriptEstudo

```bash
npm init -y
npm i typescript -D
```
Instalando o ts-node e Code Runner
```bash
npm i ts-node -D
```
Instalar run via extension

Criar dir .vscode para aplicar uma configuração personalizada
- Add settings.json
- Add dentro no settings
```javascript
"code-runner.executorMap": {
	"typescript": "npx ts-node --files"
	}
```
	
Aplicar o eslint
```bash
npm i eslint -D
```
